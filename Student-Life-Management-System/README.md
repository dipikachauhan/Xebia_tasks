# 📚 Student Life Management System

## Overview

The **Student Life Management System** is a digital workspace developed using **Notion** as part of a college project under the theme **"Build Anything, Learn Everything."**

The project is designed to help students organize their academic life by managing courses, assignments, schedules, wellness activities, and study resources within a single workspace.

---

## Problem Statement

Students often manage academic information across multiple applications, making it difficult to stay organized. This project provides a centralized workspace where important academic information can be managed efficiently using Notion's database and organization features.

---

## Project Objectives

* Organize academic information in one workspace.
* Explore Notion databases and relationships.
* Create an attractive and user-friendly interface.
* Demonstrate advanced Notion features.
* Explore the use of AI for improving student productivity.

---

## Features

* 📊 Interactive Dashboard

## Database Design

The project includes the following databases:

* Courses Database
* Master Tasks Database
* Assignments Database
* Exams Database
* Goals Database

These databases are connected using **Relations** to demonstrate relational data management within Notion.
